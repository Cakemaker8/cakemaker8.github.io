Clair Obscur: Expedition 33 Save Analyzer

This is a save file analyzer for the game Clair Obscur: Expedition 33. Currently being worked on.

You can find your save file in Users/AppData/Local/Sandfall/Saves; it will be named EXPEDITION#.sav, where # is the save slot that save is in (it can be 0-3). Once uploaded, tables will appear under each button, and will be filled out based on what has been acquired. Click each button to view the table.

Disclaimer: I do not have the deluxe edition, and therefore cannot guarantee that those items are listed properly. There may also be missing items and issues with the current items; I am working to make sure everything is correct, but issues may persist.

Note: Monoco starts with Chalier Combo and Stalact Punches, which are not mentioned in the table above.
