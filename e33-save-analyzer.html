Clair Obscur: Expedition 33 Save Analyzer

This is a save file analyzer for the game Clair Obscur: Expedition 33.

You can find your save file in Windows in C:/Users/[name]/AppData/Local/Sandfall/Saved/SaveGames/[SteamID]; it will be named EXPEDITION_#.sav, where # is the save slot that save is in (it can be 0-9), [name] is your computer's user's name, and [SteamID] is your SteamID (basically a long list of numbers). Once uploaded, tables will appear under each button, and will be filled out based on what has been acquired. Click each button to view the table.
