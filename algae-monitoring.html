Algae Monitoring Project

This is the main page for the Algae Monitoring Project, a senior design project completed by Madeline Potter, Luke Lansdown, Dalton Taylor, Tyler Brewer, and Ava Rittle.

Algae itself is not bad, but if let overgrown, it can lead to major environmental damage. This project seeks to monitor the conditions that cause algae to overgrow, and the conditions that are measured to be indicative of a bloom are as follows:

  • Temperature: need
  • pH: need
  • Oxygen: need
  • Turbidity: need
  • Nitrogen: need
  • Spectrometer: need

These values can be compared to the ones listed when selecting the month, day, and time. If those values match the ones shown above, then there is a risk of an algal bloom occurring.


Data is collected on pH, dissolved oxygen, nitrogen, turbidity, temperature, and light spectrometer. It is collected over a period of 15 minutes every 3 hours. Select the month, day, and time you want to view below.
