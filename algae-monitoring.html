Algae Monitoring Project

This is the main page for the Algae Monitoring Project, a senior design project completed by Madeline Potter, Ava Rittle, Dalton Taylor, Luke Lansdown, and Tyler Brewer.

Algae itself is not bad, but if let overgrown, it can lead to major environmental damage. This project seeks to monitor the conditions that cause algae to overgrow, and the conditions that are measured to be indicative of a bloom are as follows (exact values will be updated later):

  • Temperature: high
  • pH: high
  • Oxygen: low
  • Turbidity: high
  • Nitrogen: high
  • Spectrometer: showing green wavelengths

These values can be compared to the ones listed when selecting the month, day, and time. If those values match the ones shown above, then there is a risk of an algal bloom occurring.

Some information about the project:

  • Every part of this project was developed by the team; nothing was outsourced
  • The boat was 3D printed and covered in flex seal
  • There are 2 Arduino Unos collecting data and giving it to a Raspberry Pi 5
  • The Raspberry Pi 5 uses 4G communication to send the data to a Google Drive folder
  • The system is self-sufficient, with a battery to power everything, and a solar pannel with a charge controller to charge the battery
  • Lots of safety features were implemented, such as marine-safe wires, sodium ion batttery, reflective stickers, and water-proofing anything that needs it
